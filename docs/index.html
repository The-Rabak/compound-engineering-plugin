Frequently Asked Questions

What changed in 4.5.0?

The headline changes are rabak-java-reviewer, constitution-guardian, a tighter code-simplicity-reviewer, and a setup skill that now detects Java stacks and uses constitution-based review baselines.

Do I still get browser automation?

Yes. The current plugin uses the agent-browser skill for browser automation. The only bundled MCP server is context7.

How do agents differ from skills?

Agents are specialist teammates you invoke directly. Skills are reusable instruction sets Claude loads when a task needs them.

Where is the full inventory?

Use the reference pages for full lists, or inspect the generated plugins/compound-engineering/README.md and plugin metadata in the repository.
