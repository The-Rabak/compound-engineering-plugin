Getting Started with Compounding Engineering

The current plugin release ships 34 agents, 27 slash commands, 26 skills, and 1 MCP server. This page focuses on the current inventory and command names used by version 4.16.0.

Installation

  1. Add the marketplace:
claude /plugin marketplace add https://github.com/The-Rabak/compound-engineering-plugin
  1. Install the plugin:
claude /plugin install compound-engineering
  1. Verify it is available:
claude /plugin list

Quick Start

# Plan work
/workflows:plan Add SSO login with audit trail

# Review work
/workflows:review latest

# Execute a plan
/workflows:work docs/plans/sso-login-plan.md

Using Agents

Call a specialist directly when you need one focused perspective.

claude agent constitution-guardian "Check this branch against repo rules"
claude agent rabak-java-reviewer "Review this JVM module"
claude agent security-sentinel "Audit the auth flow"

Using Commands

The primary workflow commands now live under the /workflows: namespace.

/workflows:constitution
/workflows:brainstorm
/workflows:plan
/workflows:debug
/workflows:work
/workflows:compound

Utility commands include /lrj, /brownfield-maintenance, /reproduce-bug, /test-browser, and more.

Using Skills

Skills load reusable guidance into Claude's context.

skill: setup
skill: agent-browser
skill: create-agent-skills
skill: gemini-imagegen

Setup defaults

The setup skill now detects Java projects and adds rabak-java-reviewer plus the baseline constitution-guardian reviewer when auto-configuring review suites.

Configuration

Context7 MCP server

If Context7 does not auto-load, add it manually:

{
  "mcpServers": {
    "context7": {
      "type": "http",
      "url": "https://mcp.context7.com/mcp"
    }
  }
}

Browser automation

Browser automation is provided by the agent-browser skill, not a bundled MCP server:

npm install -g agent-browser
agent-browser install
skill: agent-browser
