Agent Reference

The current plugin ships 34 agents across review, research, design, and workflow categories. This page mirrors the generated plugin README and current agent files, so names and counts stay aligned with version 4.15.0.

How to Use Agents

claude agent rabak-java-reviewer "Review this Java service"
claude agent constitution-guardian "Check this PR against repo rules"
claude agent security-sentinel "Audit the auth flow"

New in 4.5.0

rabak-java-reviewer and constitution-guardian are now part of the shipped review inventory, and code-simplicity-reviewer now emphasizes wrong abstractions, duplication drift, and readability regressions.
