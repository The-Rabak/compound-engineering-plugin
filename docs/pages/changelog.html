v4.15.0

2026-06-01

Added

  • session-history skill for artifact-first recall with bounded recent-session fallback.
  • /workflows:debug command for orchestrating reproduction, diagnosis, decision gates, and design escalation.

Changed

  • /reproduce-bug now centers structured evidence gathering, broader intake, and clearer environment/setup blockers.
  • systematic-debugging now enforces explicit causal chains, prediction tests, failed-fix invalidation, and design escalation criteria.
  • document-review now supports artifact-aware review modes for brainstorms, plans, architecture docs, tickets, and compounded learnings.

v4.5.0

2026-04-30

Added

  • rabak-java-reviewer - Java/JVM specialist reviewer for deep modules, clear interfaces, modern Java design, and maintainable performance-aware code.
  • constitution-guardian - Repo-governance reviewer that derives standards from governing markdown, architecture docs, and repo instructions, then raises unjustified violations as P1s.

Changed

  • code-simplicity-reviewer - Tightened around DRY failures, branching complexity, dead weight, readability regressions, and bad abstractions.
  • setup skill - Detects Java projects, configures rabak-java-reviewer, and includes constitution-guardian in baseline reviewer suites.

v4.4.0

2026-04-27

Added

  • caveman skill for ultra-compressed, technically exact responses.
  • grill-me skill for decision-tree style design interrogation.
  • ubiquitous-language skill for DDD-oriented glossary generation.

v4.3.0

2026-04-25

Added

  • /workflows:constitution command for creating and updating repo constitutions.

Changed

  • Workflow commands now thread constitution context through ideation, planning, execution, and review.
  • Generated metadata and README counts were updated to 25 commands.

v4.2.0

2026-03-21

Added

  • Execution-agent, spec-review, and quality-review prompt references for workflow orchestration.
  • systematic-debugging and finishing-branch skills.
